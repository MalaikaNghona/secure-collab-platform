## Secure Collaborative Platform

A collaborative productivity web application similar to Notion, built with
security-by-design principles.

## Goals
- Build a real-world SaaS application
- Apply secure coding practices
- Learn application security engineering workflows

## Tech Stack
- Backend: FastAPI (Python)
- Database: PostgreSQL
- Authentication: Session-based (initially)
- Security: bcrypt, RBAC, OWASP ASVS

## Project Phases
1. Core application development
2. Security hardening
3. Testing & deployment
